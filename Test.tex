\question[10] Evaluate and simplify the difference quotient, \(\displaystyle{\frac{f(x+h)-f(x)}{h}}\), for the function.\\ \\ \(\displaystyle{ f(x) = x^{2} + 1 }\)
\vspace*{\stretch{1}}\answerline
\begin{solution}
\(f(x+h) = 1(x+h)^2 + 1\) 

 \(= 1(x^2 + 2xh + h^2) + 1\) 

 \(= h^{2} + 2 \, h x + x^{2} + 1\) 

 \(\displaystyle{\frac{f(x+h)-f(x)}{h} = \frac{ (h^{2} + 2 \, h x + x^{2} + 1) - (x^{2} + 1) }{h}}\) 

 \(=\displaystyle{ \frac{ h^{2} + 2 \, h x }{h}}\) 

 \(= \displaystyle{ \frac{ h(h + 2 \, x) }{h}}\) 

 \(\boxed{ h + 2 \, x }\)
\end{solution}

\(\text{Area} = \sqrt{89.0000000000000(32.0000000000000)(56.0000000000000)(1.00000000000000)}\)

\(\text{Area} \approx 399.36\)
\end{solution}
